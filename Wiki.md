## 1. Library Usage

### 1.1 Library Types

Pure 2 supports importing libraries from the following sources, with the precedence as following:

1. Pure 2 Standard Libraries (though aliases).
2. (Parcel Package): By name.
3. NuGet: By name.
4. Specific file DLL.

It's the same syntax: `Import(Library)`.
